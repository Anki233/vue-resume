# vue-resume
based on vue3.0

#### 预览地址
https://vue-resume-demo.netlify.app/

#### 安装

#### 项目地址：

（`git clone`）

```shell
git clone https://github.com/Anki233/vue-resume.git
```

#### 通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```shell
npm install
```

#### 启动服务(http://localhost:8080)

```shell
npm run serve
```

#### 发布代码

```shell
npm run build
```
